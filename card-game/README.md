  - Game 시작, 끝
  - 처음에 Card(이모지)들이 뒷면 처럼 보이게 하기
  - Card에 마우스 올리면 Over 되도록
  - Card 1 선택, Card 2 선택 시 참, 거짓 판단
    - 거짓이면, Life -1
    - Life 0이면, Game 끝
    - 모든 Card 판정 끝내면, Clear
